# Product Skill

## Scope

Use this skill for product vision, roadmap, MVP slicing, acceptance criteria, and release readiness.

## Default Focus

- Keep scope aligned with the cookbook product described in `AGENTS.md`
- Prefer small MVP increments over broad platform design
- Convert feature ideas into testable acceptance criteria

## Deliverables

- Feature brief with goal, users, and constraints
- MVP or backlog prioritization notes
- Release checklist or gating notes when needed
