# Security Skill

## Scope

Use this skill for XSS and CSRF protections, auth hardening, rate limiting, and secure media handling.

## Guardrails

- Server-side HTML sanitization is mandatory
- External image hotlinking is not supported
- Upload handling must include MIME validation, size limits, normalized filenames, storage outside webroot, optimized variants, and antivirus scanning
- Plan for secure password hashing, email verification, OAuth handling, CSRF protection, and rate limiting

## Deliverables

- Threat or control checklist
- Auth and upload security requirements
- Security testing recommendations
