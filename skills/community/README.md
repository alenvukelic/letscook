# Community Skill

## Scope

Use this skill for contributor docs, review expectations, issue triage, and onboarding guidance.

## Guardrails

- Keep contributor guidance separate from internal agent workflow rules
- Favor simple contribution paths while the repo is still bootstrapping
- Align review expectations with the planned product and security constraints in `AGENTS.md`

## Deliverables

- CONTRIBUTING or onboarding drafts
- PR and issue triage workflow notes
- Community policy follow-up tasks
