# DevOps Skill

## Scope

Use this skill for Ubuntu and nginx deployment planning, CI, backups, monitoring, and operational checklists.

## Guardrails

- Assume Ubuntu and nginx as the starting deployment model
- Include lint, test, and security checks in CI planning
- Keep database backup and restore procedures explicit

## Deliverables

- Deployment checklist
- CI pipeline outline
- Backup, restore, logging, and monitoring notes
